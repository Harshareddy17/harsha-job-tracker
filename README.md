# Harsha Job Tracker

A simple Python-based Job Application Tracker that allows storing and managing job application data using basic CRUD logic.

## Description

This project is a Python script that manages job applications by allowing users to add, view, update, and delete application entries. It demonstrates core programming fundamentals, including data structures, file or in-memory management, and business logic.

## Features

- Add new job application records
- View existing application entries
- Update application details
- Delete application entries
- Simple logic implemented using Python functions

## Technology Stack

- Python
- Basic data structures and control flow
